Introduction to JavaScript

JavaScript ― это интерпретируемый язык программирования общего назначения с динамической типизацией данных.

Интерпретируемый ― значит, что программа выполняется в исходном виде, т.е исходный код не преобразуется в машинный или промежуточный код. В связи с этим ошибок на этапе компиляции и компановки отсутствуют, есть только ошибки на этапе выполнения. Поэтому интерпретатор JavaScript можно установить на любую систему. Это дает хорошую кроссплатформенность. JavaScript также иногда используют для разработки мобильных приложений для Android. Его часто используют в тестировании для написания скриптов. Но несмотря на все это чаще всего JavaScript используется для разработки Web - приложений, а именно FrontEnd части. Следовательно основным и самым распространенным интерпретатором JavaScript является самый обычный браузер.

Динамическая типизация ― говорит о том, что тип переменных и констант определяется во время присваивания, по типу значения справа (r-value). Это в свою очередь означает, то что у переменной нет как-такогого постоянного типа данных. Например, в переменную, хранящую число, можно записать строку, и она поменяет тип. Достигается благодаря тому, что все типы являются объектами, и полиморфизму.

Переменные и типы данных

Переменные и константы

var ― Объявляет глобальную переменную независимо от области видимости, в которой она объявлена. Если записать значение в несуществующую переменную, то это будет аналогично ключевому слову var;
let ― Объявляет локальную переменную только для той области видимости, в которой она используется;
const ― Объявляет константу. Могут быть только локальными.

  1. Примитивные типы:

    • Number ― вещественное (дробное число), которое является полным аналогом типа double. Все числовые литералы неявно преобразуются в number;
