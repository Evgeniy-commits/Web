Списки

Список определений

Позволяет создавать список терминов с их определением

<dl> ― Definition List;
<dt> ― Description Term;
<dd> ― Description Details;
<dt> и <dd> всегда используются парой.
Нумерованный список
Позволяет нумеровать элементы текста. Это необходимо для перечисления в сторогом порядке.
Маркированный список
Позволяет нумеровать элементы текста. Это необходимо для перечисления в любом порядке.
Список определений
Позволяет нумеровать элементы текста и давать им определения.

Нумерованный

<ol> ― Order List;
<li> ― List Item;
  1. List Item 1
  2. List Item 2
  3. List Item 3
  4. List Item 4
  5. List Item 5

При необходимости список можно прервать и продолжить позже

  1. Продолжение первого списка
  2. Продолжение первого списка
  3. Продолжение первого списка
  4. Продолжение первого списка
  5. Продолжение первого списка

Маркированный

<ul> ― Unorder List;
<li> ― List Item;
  • List Item 1
  • List Item 2
  • List Item 3
  • List Item 4
  • List Item 5
